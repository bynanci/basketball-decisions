# Player Workflow

## Primary goal
Practice role-specific decisions and connect feedback to evidence.

## Route path
`/start` → `/training` → `/situations` → `/projects/:projectId/quiz/:promptId` → `/player-value`

## Inputs
- Role profile
- Situation and quiz prompt artifacts
- Player value summaries/evidence

## Outputs
- Decision reps completed
- Role-aware feedback context
- Optional execution feedback review

## If blocked
- Reset role via `/start`
- Load sample project via `/`
- Ask coach/analyst to create quiz prompts
