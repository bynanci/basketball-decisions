# Coach Workflow

## Primary goal
Turn analyzed possessions into teachable actions and measurable practice follow-through.

## Route path
`/reports/coach` → `/player-value` (or detail) → `/drills` → `/practice-plans` → `/practice-executions`

## Inputs
- Coach report artifacts
- Player value outputs
- Drill recommendations/catalog
- Practice execution feedback

## Outputs
- Prioritized coaching themes
- Practice plan updates
- Execution feedback review loop
