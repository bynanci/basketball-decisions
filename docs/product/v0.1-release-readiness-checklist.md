## How to use this checklist

- Allowed statuses per item: **PASS**, **FAIL**, **PARTIAL**, **NOT APPLICABLE**.
- Every line item must have an **Owner** and **Notes** before final sign-off.
- **Go/No-Go rule:**
  - **GO** only if all **Go/No-Go criteria** items are **PASS**.
  - **NO-GO** if any **Go/No-Go criteria** item is **FAIL**.
  - If any **Go/No-Go criteria** item is **PARTIAL**, release requires explicit written exception in Notes with approver name/date.
